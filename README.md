Technically, the system is powered by the MERN Stack (MongoDB, Express.js, React, and Node.js) using JavaScript. To enhance functionality and performance, we integrated a suite of modern tools:
•	Upstash is utilized for serverless data handling and caching to ensure low latency.
•	Cloudinary manages our image storage, ensuring high-quality product photos load instantly.
•	Stripe provides a secure gateway for seamless payment processing.
